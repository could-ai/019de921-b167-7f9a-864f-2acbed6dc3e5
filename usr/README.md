# Clinic Patient App

A complete Flutter application for patients to interact with their clinic. The application provides an intuitive interface for patient registration, viewing clinic services, interacting with doctor profiles, reviewing medical reports, and processing payments.

## Features

- **Patient Registration**: Users can register by providing their Name, NIC, Date of Birth, Address, and Contact Information.
- **Home / Dashboard**: Displays all available clinic services and a comprehensive list of doctors.
- **Doctor Details**: 
  - Comprehensive doctor profiles with ratings, specialties, and descriptions.
  - View services offered by the doctor.
  - Simulated live location placeholder for the doctor's clinic/office.
  - Interactive schedule calendar for booking appointments.
- **My Reports**: Patients can view their medical reports, including lab results, imaging, and prescriptions, complete with dates and the prescribing doctor's name.
- **Payments**: Integrated payment screen allowing users to view outstanding balances and choose between secure Card payments and Cash at Clinic options.

## Tech Stack

- **Framework**: Flutter
- **Language**: Dart
- **Key Packages**:
  - `google_fonts`: For customized and beautiful typography.
  - `intl`: For date and currency formatting.

## Setup and Run Instructions

1. Ensure you have the Flutter SDK installed on your machine.
2. Clone this repository.
3. Open a terminal in the project root and run `flutter pub get` to install dependencies.
4. Connect a device or start an emulator.
5. Run the app using `flutter run`.
